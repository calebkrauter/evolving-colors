Debug mode is primarily useful for smaller sizes such as 10x10. The yellow outline of the cells in debug
mode are from the back grid. This project is implmeneted using a concept known as double buffering where
one grid is used to store data and apply rules too and the second grid is used for displaying the updated
cells.
